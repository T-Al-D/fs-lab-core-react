# fs-lab-core-react

Core React frontend of the fs-lab fullstack architecture.
Webpage: https://fs-lab-core-react.onrender.com/

## Purpose

This project serves as a reference React + TypeScript frontend for experimenting
with frontend architecture, API integration, and deployment via Render.

## Tech Stack

- React
- TypeScript
- Vite

## Deployment

The application is automatically deployed via Render on every push to `main`.

## Status

Early development / experimental.
